O'zgaruvchilar faqat ular qaysi selektor ichida e'lon qilingan bo'lsa, o'sha hudud darajasida mavjud bo'ladi va o'sha hududda qo'llash mumkin. Agar ular ichki o'rnatilgan selektorlardan tashqarida aniqlangan bo'lsa, ular global miqyosda mavjud bo'ladi. Lekin, agar siz ba'zi bir joylashtirish darajasida e'lon qilingan o'zgaruvchi global miqyosda foydalanish mumkin bo'lishini istasangiz, uni maxsus yorliq bilan belgilashingiz mumkin: "!global".

    SassScript ettita asosiy ma'lumotlar turini qo'llab-quvvatlaydi:
  • raqamlar (1,2, 13, 10px)
  • matn satrlari, qo'shtirnoqli va tirnoqsiz ("foo", "bar", baz)
  • ranglar (ko'k, #04a3f9, rgba(255, 0, 0, 0.5))
  • mantiqiy qiymatlar (true, false)
  • null
  • bo'shliq yoki vergul bilan ajratilgan qiymatlar ro'yxati (1,5em 1em 0 2em; Times New Roman, Arial, sans-serif)
  • massivlar (xaritalar) (kalit1: qiymat1, kalit2: qiymat2)

SassScript boshqa barcha turdagi CSS ma'lumot turlarini ham qo'llab-quvvatlaydi, masalan, Unicode diapazonlari va "!important" deklaratsiyalar, lekin ularga murojaat qilishning maxsus usuli yo'q, ular xuddi qo'shtirnoqsiz satrlar kabi ishlatiladi.
